
# 开源前事项

## 文档

- README.md: 项目介绍，安装和使用方法
- CONTRIBUTE.md: 贡献指南，代码提交、合并规范，项目使用pylint进行代码检查
- CHANGELOG.md: 版本更新日志
- TODO.md: 项目待办事项
- LICENSE: 开源协议

## 代码

- .pylintrc: pylint代码规范配置文
- .vscode/settings.json: vscode配置文件，用户vscode下面的代码提示，可以放到contribute中，用户按照配置文件配置
- scripts: 脚本文件，如开发环境配置，打包，自动化检查等脚本等


## 其他

- ci: github actions配置文件，校验代码格式，自测，commit、merge、issue格式等
- issue template: 问题模板
- pull request template: 合并请求模板
- discussion template: 讨论模板以及欢迎词
- 代码贡献协议，用户提交代码需要签署
- 小米OAuth2.0登录需要显示logo，产品确认不需要logo
